# PdfExtractor

Separa a página que você quiser, as unifica em um único pdf e o transforma em zip.

- Necessário sempre ajustar as variáveis de busca, para que o sistema identifique se determinada página contém o que você deseja;
- Ajustar os nomes das variáveis de acordo com o que você deseja, para as deixar de forma intuitiva.
